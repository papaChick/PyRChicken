# PyRChicken
Reportcard generator made using Python.

![image](https://github.com/user-attachments/assets/ef46867d-8051-433e-84b4-7d93533fa4de)


### Instructions

1. Activate env
```
enc\Script\activate
```
2. Install requirements.txt
```
pip install -r requirements.txt
```
3. Run the program
```
python app.py
```
